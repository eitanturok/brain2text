To setup
```sh
# clone repos and symlink
git clone https://github.com/eitanturok/semantic-decoding.git
git clone https://github.com/eitanturok/brain2text.git
cd brain2text
ln -s ../semantic-decoding .

# virtual environment
uv venv --python 3.12
source .venv/bin/activate
```
